Folder structure:
- data_combined_system: This folder contains the ROS2 bags recorded during final testing, including data extracted for plotting pose estimate position of these flights.
- Data_processing: Contains the scripts used to generate plots for the report.
- ROS2_packages: Contains the ROS2 packages to estimate the payload states and to control the system with the NMPC.
- Simulation: Simulation used to validate the system model and initial NMPC parameters.
